HeliumStreamLogger

public class HeliumStreamLogger<OutputStream> : HeliumLogger where OutputStream : TextOutputStream

A HeliumLogger, that prints to a TextOutputStream.

  • use(_:outputStream:)

    Create a HeliumStreamLogger instance and set it up as the logger used by the LoggerAPI protocol.

    Usage Example:

    This example shows logging to stderr.

    struct StandardError: TextOutputStream {
    func write(_ text: String) {
        guard let data = text.data(using: String.Encoding.utf8) else {
          return
        }
        FileHandle.standardError.write(data)
    }
}

let se = StandardError()
HeliumStreamLogger.use(outputStream: se)
